SIMPLE_ARCHIVE

Simple file archiving with tar and zip support

Overview Quick Start API Reference Examples GitHub

Overview

SIMPLE_ARCHIVE provides simple file archiving capabilities for Eiffel applications. It wraps the ISE compression library to provide tar and zip archive creation and extraction.

Key Features

  • Create tar archives from files or directories
  • Create zip archives with compression
  • Extract archives to specified locations
  • List archive contents
  • Integration with SIMPLE_COMPRESSION

Quick Start

Installation

<library name="simple_archive" location="$SIMPLE_EIFFEL/simple_archive/simple_archive.ecf"/>

Environment Variable

SIMPLE_EIFFEL=/path/to/prod

Basic Usage

local
    archive: SIMPLE_ARCHIVE
do
    create archive.make

    -- Create a zip archive
    archive.create_zip ("backup.zip", << "file1.txt", "folder/" >>)

    -- Extract archive
    archive.extract ("backup.zip", "output/")
end

API Reference

SIMPLE_ARCHIVE

Feature Description
make Create archive helper
create_tar (path, files) Create tar archive from file list
create_zip (path, files) Create zip archive from file list
extract (archive, destination) Extract archive to destination directory
list_contents (archive) List files in archive

Examples

Creating a Backup Archive

local
    archive: SIMPLE_ARCHIVE
do
    create archive.make

    -- Create compressed backup
    archive.create_zip ("project_backup.zip", <<
        "src/",
        "tests/",
        "docs/",
        "README.md"
    >>)
end
